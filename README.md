# Web Scraper 

This is a simple hotel data scraper built using Go and [chromedp](https://github.com/chromedp/chromedp).

## Features
- Scrapes hotel data from a web page
- Uses headless Chrome via chromedp
- Simple and extensible structure

## How to Run

```markdown
go run main.go
```

## Requirements
- Go 1.20 or newer
- chromedp package
